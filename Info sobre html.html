Enciclopedia Completa de Etiquetas HTML

Lista oficial de las propiedades y estructuras del lenguaje web:

1. Las Graciosas, Raras y Obsoletas

  • <marquee>: Desplaza texto o imágenes por la pantalla automáticamente.
  • <blink>: Hacía parpadear el texto (obsoleta, los navegadores ya no la corren).
  • <bdo>: Voltea el orden de las letras para escribir al revés de forma exacta.
  • <bdi>: Aísla un texto para que idiomas que se leen al revés no desordenen la frase.
  • <wbr>: Marca invisible para sugerirle al navegador dónde cortar una palabra muy larga.
  • <ruby> y <rt>: Colocan anotaciones o subtítulos diminutos flotando arriba del texto.
  • <rp>: Define qué mostrar si el navegador antiguo no soporta las etiquetas ruby.
  • <center>, <font>, <big>, <strike>, <tt>: Estilos visuales antiguos hoy obsoletos (reemplazados por CSS).
  • <acronym>: Marcador antiguo de acrónimos (reemplazado por <abbr>).
  • <frame> y <frameset>: Dividían la pantalla en ventanas independientes para cargar webs distintas (obsoletas).

2. Estructura Base y Configuración Invisible

  • <!DOCTYPE html>: Declaración obligatoria que avisa que el archivo es HTML5.
  • <html>: El contenedor supremo que encierra todo el documento.
  • <head>: El cerebro invisible de la página; guarda configuraciones, estilos y títulos internos.
  • <body>: El cuerpo de la página; contiene todo lo que el usuario final ve en pantalla.
  • <meta>: Datos ocultos para los buscadores (idioma, palabras clave, descripción).
  • <title>: El texto exacto que aparece escrito en la pestaña superior de tu navegador.
  • <link> y <style>: Conectan o integran el diseño visual de la página (CSS).
  • <script> y <noscript>: Añaden lógica de programación (JavaScript) o avisos por si está desactivada.
  • <template> y <slot>: Guardan fragmentos de código ocultos listos para clonarse con programación.

3. Textos, Títulos y Párrafos

  • <h1> a <h6>: Títulos jerárquicos. H1 es el principal y grande; H6 es el más pequeño.
  • <p>, <br>, <hr>: Párrafos, saltos de línea obligatorios y líneas divisorias horizontales.
  • <strong> y <b>: Resaltan el texto en negrita (strong aporta importancia real, b es solo estética).
  • <em> e <i>: Aplican letra cursiva (em aporta énfasis hablado, i es solo visual).
  • <small>, <s>, <del>, <ins>: Modificadores de texto (letra chica, datos obsoletos, borrados o añadidos).
  • <sub> y <sup>: Letras chiquitas abajo (subíndice, ej: H2O) o arriba (superíndice, ej: E=mc2).
  • <mark>: Resalta el texto con un fondo de color amarillo brillante (como un marcador físico).
  • <abbr>, <q>, <blockquote>, <cite>: Herramientas para abreviaturas con descripción, citas cortas, citas largas y títulos de obras artísticas.
  • <dfn> y <time>: Marcan la definición de un término o fechas traducidas para computadoras.

4. Organización, Maquetación y Listas

  • <div> y <span>: Cajas contenedoras genéricas (div para bloques enteros, span para fragmentos dentro de un renglón).
  • <header>, <nav>, <main>, <footer>: Zonas principales de la página (cabecera, menús, contenido central y pie de página).
  • <article>, <section>, <aside>: Bloques independientes de contenido, secciones temáticas o barras laterales secundarias.
  • <address>, <hgroup>, <search>: Datos de contacto del autor, agrupador de subtítulos y cajas para buscadores.
  • <ul>, <ol>, <li>: Listas con viñetas de puntitos, listas numeradas y sus respectivos elementos internos.
  • <dl>, <dt>, <dd>: Listas de términos con sus definiciones (estilo formato diccionario).

5. Enlaces, Multimedia, Tablas y Formularios

  • <a>: Crea hipervínculos clickables hacia cualquier otra página de internet.
  • <img>, <picture>, <source>, <figure>, <figcaption>: Etiquetas para meter imágenes normales, adaptables a móviles y descripciones al pie de la foto.
  • <map> y <area>: Permiten dibujar zonas de clic invisibles encima de una foto.
  • <svg> y <canvas>: Dibujos vectoriales directos por código matemático y lienzos interactivos para programar juegos.
  • <table>, <tr>, <th>, <td>, <thead>, <tbody>, <tfoot>, <colgroup>, <col>, <caption>: La familia completa para armar tablas estructuradas de datos, filas, celdas y resúmenes.
  • <form>, <input>, <textarea>, <button>, <select>, <option>, <optgroup>, <datalist>, <label>, <fieldset>, <legend>, <output>: Toda la maquinaria para recolectar datos de usuarios (casillas de texto, contraseñas, menús desplegables y botones de envío).
  • <video>, <audio>, <track>, <object>, <embed>, <param>: Reproductores nativos de cine, sonido, subtítulos cronometrados e incrustación de archivos externos como PDFs.

6. Código, Interactividad Avanzada y Conexiones

  • <code>, <pre>, <samp>, <kbd>, <var>: Formatos especiales para textos informáticos (bloques que respetan espacios, ejemplos del sistema, simulación de teclas físicas y variables matemáticas).
  • <details> y <summary>: Menús desplegables interactivos nativos sin usar programación.
  • <dialog>: Ventanas emergentes flotantes (pop-ups) que bloquean la pantalla y se cierran con la tecla 'Esc'.
  • <meter> y <progress>: Indicadores gráficos estáticos o barras animadas de procesos de carga en curso.
  • <iframe> y <portal>: Ventanas para meter páginas webs externas completas dentro de tu propio sitio.
